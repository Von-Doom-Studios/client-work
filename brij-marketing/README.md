# brij-marketing

The operating folder for Brij marketing. Self-contained operating manual. Anyone new should be able to open `01-GETTING-STARTED.md`, read it, and understand the system inside 10 minutes.

**Owner:** VD-Marketing
**Client:** Brij (https://brij.fi)
**Channels in scope:** X only.
**Cadence:** 3 X posts per week (Mon product / Wed comedy / Fri proof). Plus monthly newsletter. Newsletter is integrated to campaigns; every campaign declares its newsletter tie-in.
**Last updated:** 2026-05-26
